# Wireshark Network Traffic Analysis Lab

## Objective
The objective of this lab is to understand the basic concepts of Wireshark and network traffic analysis. 

This project focuses on capturing and analyzing network packets, including:
- Identifying common network protocols
- Analyzing TCP Three-Way Handshake
- Capturing and analyzing ICMP (Ping) traffic

  ## Background

Wireshark is a widely used network protocol analyzer that allows users to capture and inspect network traffic in real time. It is commonly used by network administrators, cybersecurity professionals, and students to analyze network behavior and troubleshoot issues.

Wireshark supports deep inspection of hundreds of protocols and provides detailed packet-level information such as IP addresses, ports, and protocol types.

In this lab, Wireshark was used to capture live network traffic and analyze different protocols and communication patterns.

## Tools Used
- Wireshark
- Command Prompt (Windows)
- Web Browser (Google Chrome)

## Packet Capture Procedure

1. Wireshark was installed and opened.
2. The active network interface (Wi-Fi) was selected.
3. Packet capture was started by double-clicking the interface.
4. Network activity was generated by browsing websites.
5. Packet capture was stopped using the red stop button.
6. The captured data was saved for analysis.
