# Network Traffic Analysis Using Wireshark

## Overview
This project demonstrates basic network traffic analysis using Wireshark.

## Tools Used
- Wireshark
- Web Browser 

## Objectives
- Capture live network traffic
- Analyze DNS requests
- Identify protocols such as TCP and HTTPS

## Findings
- Observed DNS queries from my device
- Majority of traffic used HTTPS (port 443)
- Multiple external IP connections detected

## Skills Gained
- Packet analysis
- Network monitoring
- Basic cybersecurity investigation
